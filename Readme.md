# Reporting for WPF - How to Customize the Ribbon Toolbar

This example demonstrates how you can customize the Report Designer control's toolbar. It shows how to remove standard toolbar controls and add custom ones.Â 

To accomplish this, use the Designer's [`RibbonCustomizationActions`](https://docs.devexpress.com/WPF/DevExpress.Xpf.Reports.UserDesigner.ReportDesigner.RibbonCustomizationActions) property, which specifies a collection of actions used to customize the Designer's ribbon toolbar.Â These actions allow you to remove existing items or add new items both in the designer and preview pages.
